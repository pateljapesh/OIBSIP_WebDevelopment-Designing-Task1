# OIBSIP_WebDevelopment-Designing-Task1

📝 Internship Task Details
Task Assignment: > "A landing page is one of the best web development projects for beginners. This project demands a foundational understanding of HTML and CSS. You will learn how to add columns, divide sections, arrange items, add headers, footers. Most importantly, you will use your creativity to make the page look impressive. The alignments, the padding, the color palette, boxes and all the other elements on the page require attention. Use CSS carefully to make sure the elements on the page do not overlap."


🚀 Features
Clean & Responsive Layout: Divided sections using structured HTML elements for a seamless user experience.

Navigation & Anchor Links: A functional header with active smooth-scrolling links (Home, Destinations, Packages, etc.) pointing to relevant sections.

Hero Section: An engaging hero banner with a search bar feature allowing users to simulate searching for travel destinations.

Interactive Cards: A beautifully aligned grid layout showcasing "Popular Destinations" with image hover effects, descriptive text, and clear pricing tags.

Modern UI/UX: Careful consideration given to color palettes, padding, typography, and preventing element overlaps.


🛠️ Tech Stack
HTML5: Semantics, page layout structure, and content categorization.

CSS3: Flexbox/Grid for alignments, padding/margin adjustments, hover animations, and custom styling.
