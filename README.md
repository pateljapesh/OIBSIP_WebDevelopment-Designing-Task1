# OIBSIP_WebDevelopment-Designing-Task1

📝 Internship Task Details
Task Assignment: > "A landing page is one of the best web development projects for beginners. This project demands a foundational understanding of HTML and CSS. You will learn how to add columns, divide sections, arrange items, add headers, footers. Most importantly, you will use your creativity to make the page look impressive. The alignments, the padding, the color palette, boxes and all the other elements on the page require attention. Use CSS carefully to make sure the elements on the page do not overlap."


## 🚀 Features
Clean & Responsive Layout: Divided sections using structured HTML elements for a seamless user experience.

Navigation & Anchor Links: A functional header with active smooth-scrolling links (Home, Destinations, Packages, etc.) pointing to relevant sections.

Hero Section: An engaging hero banner with a search bar feature allowing users to simulate searching for travel destinations.

Interactive Cards: A beautifully aligned grid layout showcasing "Popular Destinations" with image hover effects, descriptive text, and clear pricing tags.

Modern UI/UX: Careful consideration given to color palettes, padding, typography, and preventing element overlaps.


## 🎯 Objective
The primary goal of this project was to build a clean, creative, and fully responsive landing page from scratch. The task focuses on mastering the foundational concepts of front-end development, specifically focusing on layout design, element alignment, typography, and a cohesive color palette to ensure an optimal user experience across various screen sizes.


## 🛠️ Tools & Technologies Used
* **HTML5:** For structuring the webpage sections, headers, footers, and semantic content layouts.
* **CSS3:** For custom styling, typography, padding/margin adjustments, layout alignments, and creating interactive elements.
* **FontAwesome / Google Fonts:** For incorporating modern typography and intuitive UI icons.


## 📝 Steps Performed
1. **Wireframing & Layout Setup:** Planned the structure to include essential sections: Hero banner, Destinations, Pricing/Packages, About, and Contact.
2. **Structuring with HTML:** Built out the clean semantic code, dividing content cleanly into dynamic grid columns and logical containers.
3. **Styling & Alignment with CSS:** Applied custom layouts while keeping careful attention to margins and paddings to prevent overlapping and guarantee a pixel-perfect design.
4. **Section Building & Interactive Elements:**
   * Curated a **Popular Destinations** card layout with hover effects.
   * Created a **Travel Packages** comparison table mapping tiered pricing plans.
   * Coded a functional **Contact Us** message form with inputs for data collection.
5. **Testing & Fine-Tuning:** Checked the web page functionality, verified asset paths, and ensured smooth internal anchor link navigation.


## 🚀 Key Outcomes
* Developed a smooth, aesthetically pleasing landing page with clear layout structures.
* Successfully eliminated layout overlap errors using disciplined CSS spacing principles.
* Designed a clean form validation interface for an improved user interaction loop.
* Built confidence in translating standard layout briefs into complete, creative front-end web products.
